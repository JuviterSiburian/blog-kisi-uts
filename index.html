Memulai Perjalanan Cloud Computing

Tanggal: 19 April 2025

Cloud computing telah mengubah cara kita membangun dan men-deploy aplikasi. Dengan menggunakan layanan AWS seperti EC2, S3, dan lainnya, kita dapat dengan mudah mengelola infrastruktur tanpa perlu khawatir tentang perangkat keras fisik.

Dalam posting blog pertama ini, saya akan berbagi pengalaman saya dalam menyiapkan blog ini menggunakan Amazon EC2 sebagai server dan Amazon S3 untuk penyimpanan.

Mengoptimalkan Aplikasi Web di AWS

Tanggal: 18 April 2025

Ada banyak cara untuk mengoptimalkan aplikasi web yang berjalan di AWS. Mulai dari pemilihan jenis instance yang tepat, menggunakan Auto Scaling Groups, hingga mengimplementasikan Content Delivery Network dengan Amazon CloudFront.

Blog ini sendiri menggunakan instance EC2 sederhana, tetapi tetap dapat dioptimalkan lebih lanjut seiring dengan meningkatnya kebutuhan.
